# 🏥 Nursing School Planner App

A comprehensive productivity and tracking system designed specifically for nursing students to manage their academic workload, clinical experiences, exam preparation, and wellness.

## 📋 Features

### 1. **Dashboard** 
- Auto-pulls statistics from all other sheets
- Overall GPA summary
- Assignment completion status
- Exam preparation progress
- Clinical hours tracker
- Habit completion rates
- Self-care check-ins completed

### 2. **Calendar**
- Monthly/weekly view
- Mark assignment due dates
- Class schedules
- Exam dates
- Clinical shift schedules
- Color-coded event types
- Quick event details on hover

### 3. **Grade Tracker**
- Track all course grades
- Color-scale conditional formatting (red to green)
- Weighted GPA calculation
- Course weights
- Letter grade display
- Semester GPA summary
- Cumulative GPA tracking

### 4. **Assignments**
- Assignment name and description
- Due date tracking
- Subject/course
- Status (Not Started, In Progress, Completed)
- Grade percentage
- Overdue counter formula
- Grade % auto-calculation
- Priority levels

### 5. **Clinical Log**
- Shift date and time
- Preceptor name
- Unit/Department
- Patient count
- Skills practiced
- Procedures performed
- Reflections/Notes
- 50+ session rows
- Hours calculation

### 6. **Skills Checklist**
- 40+ nursing skills
- Completion status (Not Started, In Progress, Mastered)
- Date first practiced
- Last practiced date
- Confidence level (1-5)
- Comments
- Category organization

### 7. **NCLEX Prep**
- Question category
- Topic
- Difficulty level
- Correct/Incorrect tracking
- Accuracy rate formulas by category
- Category sidebar with progress
- Time spent per topic
- Weak area identification

### 8. **Weekly Planner**
- 7-day time-slot grid
- Color-coded by activity type (Class, Study, Clinical, Personal)
- Study blocks
- Class schedule
- Clinical shifts
- Break times
- Legend for color key

### 9. **Exam Tracker**
- Exam name and date
- Subject
- Study progress (%)
- Practice test scores
- Target score vs actual
- Days until exam countdown
- Topics to review
- Study materials

### 10. **Habit Tracker**
- Daily habits (Exercise, Sleep 8hrs, Meditate, Journal, etc.)
- Weekly tracking grid
- Completion percentage
- Streak counter
- Motivational notes
- Category (Health, Wellness, Academic)

### 11. **Self-Care Log**
- Self-care activities completed
- Date and time
- Activity type (Sleep, Exercise, Meditation, Social, Nutrition, etc.)
- Duration
- Mood before and after
- Notes
- Monthly self-care hours

### 12. **Class Schedule**
- Course name
- Meeting times
- Location
- Instructor
- Office hours
- Contact information
- Course prerequisites/requirements

## 📊 Color Coding System

- 🔴 **Red**: Overdue, Low grades, Not Started
- 🟡 **Yellow**: In Progress, Warning
- 🟢 **Green**: Completed, On Track, Good Grades
- 🔵 **Blue**: Clinical, Important Info
- 🟣 **Purple**: Personal/Self-Care
- ⚫ **Gray**: Not applicable

## 🎯 How to Use

1. Start with the **Dashboard** for your overview
2. Update your **Class Schedule** first
3. Add **Assignments** as they're assigned
4. Track **Clinical Experiences** in real-time
5. Monitor **Grades** as they come in
6. Use **Weekly Planner** for time management
7. Track **Habits** and **Self-Care** daily
8. Review **NCLEX Prep** progress regularly
9. Check **Exam Tracker** for test prep status
10. Reference **Calendar** for all important dates
11. Update **Skills Checklist** after clinical practice

## 🚀 Getting Started

- All sheets are linked for automatic calculations
- Use dropdown menus for consistency
- Follow the color-coding system
- Update daily for best results
- Review dashboard weekly
